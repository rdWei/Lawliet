##
 ### Installation
  ```
  # clone the repository
  $ git clone https://github.com/rdWei/Lawliet
  
  # change the working directory to Lawliet
  $ cd Lawliet

  # install the requirements
  $ python3 -m pip install -r requirements.txt
  ```
##
### Usage
  ```
  //Linux
  bash lawliet.sh
  
  //Windows
  cd cmd
  python main.py
  ```
